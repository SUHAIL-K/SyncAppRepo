Performance Evaluation System

## Overview

Sync App is a staff performance evaluation and monitoring system developed for internal organizational use. The platform was designed to streamline employee performance tracking, staff activity monitoring, reporting, and centralized workflow management through both mobile and web applications.

The project includes:

* Mobile application built using Flutter
* Web application built using React.js and Next.js
* Cloud-based backend infrastructure using Firestore and Supabase
* Real-time synchronization and centralized reporting systems

> Note: This repository is a project showcase and documentation repository. Source code and live demo access are restricted due to company confidentiality and official organizational data.
